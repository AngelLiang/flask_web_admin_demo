# flask web admin demo

flask web 后台模板

主要基于以下模块：

- flask-user
- flask-admin
- start-bootstrap-admin-2

# 快速开始

```PowerShell
python3 -m venv venv
.\venv\Scripts\activate
(venv) pip3 install -r .\requirements.txt
flask initdb
flask createsuperuser
flask run
```
